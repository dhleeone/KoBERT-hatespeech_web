* pip install mxnet

* pip install git+https://git@github.com/SKTBrain/KoBERT.git@master
* pip install xlrd
* pip3 install torch==1.10.0+cpu torchvision==0.11.1+cpu torchaudio==0.10.0+cpu -f https://download.pytorch.org/whl/cpu/torch_stable.html
